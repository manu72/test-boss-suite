Getting Started

Getting started with Test Boss is quick and easy. Follow these simple steps to begin your testing journey:

  1. Installation:
    Open your terminal and run:
    npm install -g test-boss
  2. Create Your First Test:
    Create a new file called example.test.js and add:
    import { test } from 'test-boss'; test('my first test', () => { expect(2 + 2).toBe(4); });
  3. Run Your Tests:
    In your terminal, run:
    test-boss example.test.js

Features Deep Dive

Test Boss comes packed with features that make testing accessible and enjoyable for beginners:

Intelligent Test Runner

Our smart test runner automatically detects and runs your tests, providing clear feedback about what passed and what needs attention.

// Example of Test Boss's clear feedback test('check user authentication', () => { const user = login('test@example.com', 'password'); expect(user.isAuthenticated).toBe(true); expect(user.email).toBe('test@example.com'); });

Assertion Library

Write expressive tests with our built-in assertion library that uses natural language patterns:

expect(response.status).toBe(200); expect(user.name).toContain('John'); expect(items).toHaveLength(3); expect(callback).toHaveBeenCalled();

Async Testing

Testing asynchronous code is straightforward with Test Boss:

test('async operations', async () => { const data = await fetchUserData(); expect(data.status).toBe('success'); });
