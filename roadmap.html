Product Roadmap

Our journey to simplify software testing

Our Vision

Test Boss Suite aims to democratize software testing by providing intuitive, powerful testing tools for developers at the beginning of their journey. We believe that good testing practices should be accessible to everyone.

Development Phases

Phase 1: Foundation (Q1-Q2 2024)

Test Boss Browser:

  • Basic page navigation and element interaction
  • Screenshot comparison capabilities
  • Record-and-playback functionality
  • Puppeteer integration

Phase 2: Lightning Web Component Focus (Q3 2024)

  • Component rendering utilities
  • Event simulation tools
  • Jest integration
